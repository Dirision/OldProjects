Introduction:
Hello and thank you for trying Building Blocks! Building Blocks is an 
interactive game for kids to help them further develop their ability
to differentiate, and identify, shapes and colors. 

Developer Info
Developed by Nick DiRisio, this project was build in Java using Swing, AWT, and
Java sound.  

HOW TO PLAY:

BUILD MODE:
1: Click features from the drop down menu
2: The shape will draw instantly
3: The shape will draw each time a different feature is selected
4: Click the switch to quiz mode button to switch to quiz mode 

QUIZ MODE:
1: Guess by selecting options from the two buttons
2: Check your answers by clicking the "Check" Button
3: Your score is shown at the top of the window
4. Switch to build mode by clicking the switch button

NOTE:
Yes, there is an easter egg of my previous work terribly hidden in the opening
